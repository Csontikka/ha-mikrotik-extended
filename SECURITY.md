# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest release | Yes |
| Older versions | No |

Only the latest release receives security updates. Please upgrade before reporting.

## Reporting a Vulnerability

**Do not open a public issue for security vulnerabilities.**

Instead, use [GitHub's private vulnerability reporting](https://github.com/Csontikka/ha-mikrotik-extended/security/advisories/new) to report security issues confidentially.

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

You will receive an initial response within 72 hours. Once confirmed, a fix will be released as soon as possible with credit to the reporter (unless anonymity is requested).

## Scope

This policy covers the MikroTik Extended integration for Home Assistant. Issues in upstream dependencies should be reported to their respective projects.
